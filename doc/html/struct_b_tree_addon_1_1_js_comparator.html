Detailed Description

Functor that provides a strict weak ordering for JsHandle keys.

Ensures a consistent, total ordering of JavaScript values in the B+Tree:

  • If both handles wrap numbers, compares their stored numberVal.
  • Otherwise falls back to V8â€™s comparison APIs on the underlying Value.
